# RSP-Style Release Safety Process (Simplified)

This document provides a **lightweight Risk & Safety Process** similar to frontier labs.

## Stage Gates

### Gate A — Pre-training kickoff
- Threat model approved
- Dataset legal review complete
- Sandbox plan approved
- Incident response plan exists

### Gate B — Mid-training checkpoint
- Capability uplift evaluation
- Jailbreak regression suite
- Data contamination scan

### Gate C — Post-training checkpoint
- RLVR reward hacking audit
- Tool-use restrictions verified
- External red-team scheduled

### Gate D — Pre-release
- Full eval suite run
- Model card draft complete
- Mitigations deployed
- Monitoring dashboards ready

## Required Artifacts
- System card
- Eval report
- Red-team report
- Safety mitigations spec
- Rollback procedure
